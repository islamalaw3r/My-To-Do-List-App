📝 My To-Do List App
A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript as part of Web Development 1 course (Assignment 9 – Week 09).

---

##🚀 Features
➕ Add new tasks easily
☑ Mark tasks as completed
🗑 Delete individual tasks
🔍 Filter tasks (All / Active / Completed)
🧹 Clear all completed tasks
📊 Live counter for remaining tasks
🎨 Clean UI with olive-green theme and modern design

---

##🛠️ Technologies Used
-HTML5
-CSS3
-JavaScript (Vanilla JS)

---

##📸 Preview
A minimal and modern to-do list app with olive green styling and interactive task management.

---

##🎯 How It Works
-Type your task in the input field
-Click Add
-Click on task text to mark it as completed
-Use filters to view tasks by status
-Use Clear Completed to remove finished tasks
